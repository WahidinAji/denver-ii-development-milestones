# WHAT IS THIS?
this is a DENVER II development milestones, read this document for more details 
- http://www.paedsportal.com/application/files/7314/5308/0217/Denver_II_Developmental_Milestones.pdf
- https://www.dchealthcheck.net/trainings/documentation/epsdt/development.pdf

in short, there are 4 pillars of that development milestones:
- Personal Social
  getting alone with people and caring for personal needs
- Fine Motor Adaptive
  eye hand coordination, manipulation of small objects, and problem solving
- Gross Motor
  walking, running, jumping, and other large muscle movements
- Communication or Language
  Hearing, understanding, speaking or using language

# EXPECTATIONS APP
I can create a list of developmental milestones for a child based on their age.But not like, parents create based on milestones. But parents create based on in which date and age the children have reached milestones based on development milestones (DENVER II).
